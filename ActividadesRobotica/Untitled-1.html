Actividad 1: Contador con botones

Código de la actividad:

// Código de la actividad 1: Contador con botones
input.onButtonPressed(Button.A, function () {
    if (count < 10) {
        count += 1
    }
    basic.showNumber(count)
})
input.onButtonPressed(Button.AB, function () {
    basic.showNumber(0)
    count = 0
})
input.onButtonPressed(Button.B, function () {
    if (count > 0) {
        count += 0 - 1
    }
    basic.showNumber(count)
})
let count = 0
basic.showNumber(0)

Este código permite que el robot cuente con los botones A y B. Cuando presionas el botón A, el contador aumenta. Si presionas el botón B, el contador disminuye. Si presionas el botón AB, el contador se reinicia a 0.

Actividad 2: Control de volumen con gestos

Código de la actividad:

// Código de la actividad 2: Control de volumen con agitar
input.onGesture(Gesture.Shake, function () {
    volumen = randint(1, 3)
    music.setVolume(volumen)
    if (volumen == 1) {
        basic.showIcon(IconNames.Square)
    } else if (volumen == 2) {
        basic.showIcon(IconNames.Scissors)
    } else {
        basic.showIcon(IconNames.Chessboard)
    }
    basic.pause(1000)
    basic.clearScreen()
    music.setVolume(127)
})
let volumen = 0
basic.clearScreen()
music.setVolume(0)

En este código, el robot ajusta el volumen del sonido al agitarlo. Al cambiar el volumen, también muestra un icono en la pantalla. Los iconos representarán diferentes niveles de volumen: un cuadrado, unas tijeras o un tablero de ajedrez. El volumen vuelve a su valor por defecto después de 1 segundo.
